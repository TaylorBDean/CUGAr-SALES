# Setup Wizard - Quick Start

## Running the Wizard

```bash
# From project root
python3 -m cuga.frontend.setup_wizard

# Or directly
python3 src/cuga/frontend/setup_wizard.py
```

## What It Does

The setup wizard provides an interactive CLI interface for:

1. **Capability Showcase** - Explains what the platform does
2. **Adapter Overview** - Shows all available data sources
3. **Credential Configuration** - Securely enter API keys
4. **Connection Testing** - Validate credentials work
5. **Config Saving** - Saves to `.env.sales` file

## Features

- **Clean UI** - Color-coded, easy to read
- **Secure Input** - Passwords/secrets use getpass (hidden input)
- **Validation** - Tests connections before saving
- **Existing Config** - Loads and preserves existing credentials
- **Flexible** - Configure all adapters or just specific ones

## After Setup

Once configured, use your credentials:

```bash
# Load environment
source .env.sales

# Test all adapters
python3 scripts/setup_data_feeds.py

# Or test with Python
python3 -c "
from cuga.adapters.sales.factory import create_adapter
adapter = create_adapter('salesforce', trace_id='test')
accounts = adapter.fetch_accounts({'limit': 5})
print(f'Fetched {len(accounts)} accounts')
"
```

## Configuration File

The wizard creates `.env.sales`:

```bash
# CUGAr-SALES External Data Adapter Configuration
# Generated by Setup Wizard
# Date: 2026-01-04T...

# Salesforce
SALES_SALESFORCE_ADAPTER_MODE=live
SALES_SALESFORCE_CLIENT_ID=3MVG9...
SALES_SALESFORCE_CLIENT_SECRET=<secret>
SALES_SALESFORCE_USERNAME=user@company.com
SALES_SALESFORCE_PASSWORD=<password>
SALES_SALESFORCE_INSTANCE_URL=https://mycompany.my.salesforce.com

# HubSpot
SALES_HUBSPOT_ADAPTER_MODE=live
SALES_HUBSPOT_API_KEY=<private-app-token>
```

## Security Notes

- Credentials stored in `.env.sales` (add to `.gitignore`)
- Use environment variables (not hardcoded)
- Secrets hidden during input (getpass)
- No obfuscation - plain text for transparency

## Troubleshooting

**Import Error:**
```bash
export PYTHONPATH=/path/to/CUGAr-SALES/src:$PYTHONPATH
python3 -m cuga.frontend.setup_wizard
```

**Connection Test Fails:**
- Verify credentials are correct
- Check API quotas and rate limits
- Ensure network connectivity
- Review adapter-specific setup guides
